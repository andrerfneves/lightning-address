# Get an Address

The easiest way to get a Lightning Address is to use a wallet or service that provides one automatically.

## Recommended Wallets

These wallets give you a Lightning Address when you sign up:

### ZBD
- **Address format:** `username@zbd.gg`
- **Platforms:** iOS, Android, Web
- **Features:** Full LNURL-pay support including comments, sender identity, and payment verification
- [Get ZBD](https://zbd.gg)

### Wallet of Satoshi
- **Address format:** `username@walletofsatoshi.com`
- **Platforms:** iOS, Android
- **Features:** Simple, beginner-friendly
- [Get Wallet of Satoshi](https://walletofsatoshi.com)

### Strike
- **Address format:** `username@strike.me`
- **Platforms:** iOS, Android, Web
- **Features:** USD and BTC balances, bank integration
- [Get Strike](https://strike.me)

### Alby
- **Address format:** `username@getalby.com`
- **Platforms:** Browser extension, Web
- **Features:** Web-native, browser extension, Nostr integration
- [Get Alby](https://getalby.com)

## Custom Domains

Want a Lightning Address on your own domain? You have options:

1. **Self-hosted** — Run your own LNURL server ([see guide](/docs/setup/self-hosted))
2. **Bridge servers** — Connect your existing Lightning node ([see guide](/docs/setup/bridge-servers))
3. **Hosted services** — Use a provider that supports custom domains

## Business Use

If you're a business looking to accept Lightning payments:

- **ZBD** offers business APIs and custom domain support
- **BTCPay Server** provides self-hosted payment processing
- **LNbits** offers extensible Lightning tooling

Check the [Directory](/directory) for a full list of providers and their capabilities.
